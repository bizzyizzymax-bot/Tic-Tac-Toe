# Tic-Tac-Toe
A Frontend Tic Tac Toe game that allows two users to play on the same device.

## Live Demo
https://tictactoeproject-im.netlify.app/

## Features
- A board that allows players to place either an X or O depending on their chose player type.
- Text that show the status of the players' turns.
- An alert that shows which player has won the game

## Tech Stack
- Html
- CSS
- Javascript
